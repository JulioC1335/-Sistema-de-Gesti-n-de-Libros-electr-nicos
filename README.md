# -Sistema-de-Gesti-n-de-Libros-electr-nicos
Este es un proyecto individual construido en el lenguaje Go (Golang), cuyo objetivo es facilitar la administración de una biblioteca digital. Permite registrar, consultar, modificar y eliminar libros electrónicos, almacenando sus datos de forma estructurada y accesible.  El sistema está diseñado bajo un enfoque de programación funcional.
